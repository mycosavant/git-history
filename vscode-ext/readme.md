# Git File History

Quickly browse the history of a file from any git repository

![feature X](https://user-images.githubusercontent.com/1911623/52807635-d5043480-306a-11e9-9b03-351b7cda4936.gif)

## Requirements

- VSCode 1.74.0 or later
- Git installed and available in PATH

## Installation

Install from the VSCode marketplace or build from source.

### Building from Source

```bash
cd vscode-ext
npm install
npm run build
```

## Usage

1. Open any file in a git repository
2. Run command `Git File History` from the Command Palette (Ctrl+Shift+P)
3. Browse the history in the webview panel
